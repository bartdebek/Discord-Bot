# Discord-Bot

This is a Discord app bot that has couple of functions:
!quiz - draws a random question from JSON file, awaits 60 seconds an answer, if player answers correctly they are given a point and are put on a leaderboard
!leaderboard - shows a current quiz leaderboard
!konf - sends a random Confucius quote (in Polish)
!poz - sends a positive quote to cheer you up (in Polish)
