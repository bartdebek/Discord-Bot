# Discord-Bot

<b>This is a Discord app bot that has couple of functions:</b>

<b>!quiz</b> - draws a random question from JSON file, awaits 60 seconds for an answer, if player answers correctly they are given a point and are put on a leaderboard<br>
<b>!leaderboard</b> - shows a current quiz leaderboard<br>
<b>!konf</b> - sends a random Confucius quote (in Polish)<br>
<b>!poz</b> - sends a positive quote to cheer you up (in Polish)<br>
